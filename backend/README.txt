# SonicWrite

AI-powered content and resume assistant focused on fast, structured output and automation.

## 🚀 Live App

https://sonicwrite.sonicresume.com

## ✨ Features

* AI content generation
* Resume and structured writing tools
* Fast response generation
* Clean UX for productivity workflows

## 🔐 Authentication

* Full login/signup system

## 💳 Payments

* AI: OpenAI API
